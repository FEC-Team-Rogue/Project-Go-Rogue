# Project-Revamp
This is the front end of a retail website

## Building Client

First install dependencies:

```sh
npm install
```

To create a production build:

```sh
npm run build
```

To create a development build:

```sh
npm run build-dev
```

## Running

```sh
npm start
```

## Testing

To run unit tests:

```sh
npm test
```
