# Project-Revamp
This is the front end of a retail website

## Set up the Client

First install dependencies:

```sh
npm install
```

To create a production build:

```sh
npm run build
```

Run server:

```sh
npm start
```

Type in browser:

```sh
http://localhost:3000
```
